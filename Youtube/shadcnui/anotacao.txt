Lista de comandos mais importantes usados:
*npx create-next-app@latest aula (cria projeto nextjs)
*npx shadcn@latest init (instala o shadcn/ui no projeto)
*yarn run dev (rodar o projeto nextjs)
*npx shadcn@latest add input (adicionar o componente input ao projeto seguindo o: https://ui.shadcn.com/docs/components/input)

Passos da aula do shadcn/ui Sujeitoprogramador:
1- instalei o nextjs e depois fui instalando os componentes do shadcn/ui (entre 0 a 10 minutos de video)

2- Criei o menu lateral (primeiro irei fazer MOBILE FIRST) (10 minutos ate 26 minutos do video)
Criei a pasta sidebar e index.tsx

3- Criei a sidebar do desktop (26 minutos ate 35 minutos do video)
Ainda dentro do index.tsx criei a tag aside

4-Criei a dashboard (35 minutos ate 51 minutos do video)
*Dentro de page.tsx eu criei varios cards
*Instalei o charts do shadcn/ui para fazer o grafico
*Atualizei o global.css para fazer o grafico funcionar
*Criei a pasta chart e o arquivo index.tsx, dentro dele criei o grafico. O nome dele eh ChartOverview
*Adicionei o <ChartOverview/> (grafico/componente/chart) no arquivo page.tsx e pronto
Resumindo: o <ChartOverview/> eh a juncao tanto do component chart como o component card entre outros. Assim o chart/grafico ficou dentro de um card personalizado, formando um card com grafico.

5-criei o card de ultimos clientes (51 minutos ate 58 minutos do video)
*Criei a pasta sales dentro da pasta de components, essa pasta tem o arquivo index.tsx.
Assim como anterior, criei um card e dessa vez com uma lista de ultimos clientes.
