-----------------------------------------------------------------------------------------------------
Para criar um projeto React
1-no terminal digitar:
    npx create-react-app nomeDoProjeto
2- y (para aceitar o processeguir)

obs: excluir arquivos que nao irei utilizar dentro desse projeto react
-----------------------------------------------------------------------------------------------------
Para abrir um servidor local para abrir o site
Dentro da pasta do arquivo site ou pasta do react do site, executar o comando:
-npm run start

------------------------------------------------------------------------------------------
USANDO YARN

1- No terminal do projeto/pasta, digitar:
yarn creacte react-app nomeDoProjeto

2- Abrir servidor com YARN, digitar:
yarn start
